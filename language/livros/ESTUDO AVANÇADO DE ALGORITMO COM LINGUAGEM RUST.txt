ESTUDO AVANÇADO DE ALGORITMO COM LINGUAGEM RUST



Para fechar o teu arsenal com chave de ouro, vamos à linguagem que é a sensação
do momento para sistemas seguros e de alta performance: Rust.

O Rust é conhecido por ser "implacável" com a memória, o que o torna perfeito para
implementar os algoritmos do Cormen, pois força-te a entender exatamente quem é
o "dono" de cada dado (Ownership).

Aqui estão os 15 algoritmos em Rust:

2. Árvore Binária de Busca (BST - Inserção)

Rust
fn tree_insert(node: &mut Option<Box<Node>>, key: i32) {
    match node {
        Some(ref mut n) => {
            if key < n.key { tree_insert(&mut n.left, key); }
            else { tree_insert(&mut n.right, key); }
        }
        None => {
            *node = Some(Box::new(Node { key, left: None, right: None }));
        }
    }
}
3. Max-Heap (Heapify)

Rust
fn max_heapify(a: &mut [i32], i: usize, heap_size: usize) {
    let l = 2 * i + 1;
    let r = 2 * i + 2;
    let mut maior = i;
    if l < heap_size && a[l] > a[i] { maior = l; }
    if r < heap_size && a[r] > a[maior] { maior = r; }
    if maior != i {
        a.swap(i, maior);
        max_heapify(a, maior, heap_size);
    }
}

5. Busca em Largura (BFS)

Rust
fn bfs(grafo: &HashMap<i32, Vec<i32>>, s: i32) {
    let mut q = VecDeque::new();
    let mut dist = HashMap::new();
    q.push_back(s);
    dist.insert(s, 0);
    while let Some(u) = q.pop_front() {
        if let Some(vizinhos) = grafo.get(&u) {
            for &v in vizinhos {
                if !dist.contains_key(&v) {
                    dist.insert(v, dist[&u] + 1);
                    q.push_back(v);
                }
            }
        }
    }
}

    if a.len() <= 1 { return; }
    let p = partition(a);
    quicksort(&mut a[0..p]);

9. Algoritmo de Kruskal (MST)

Rust
fn kruskal(edges: &mut Vec<Edge>, n: usize) -> Vec<Edge> {
    edges.sort_by_key(|e| e.weight);
    let mut ds = DisjointSet::new(n);
    edges.iter().filter(|e| ds.union(e.u, e.v)).cloned().collect()
}

fn bellman_ford(edges: &[Edge], n: usize, s: usize) -> Option<Vec<i32>> {
    let mut d = vec![i32::MAX; n];
    d[s] = 0;
    for _ in 0..n-1 {
        for e in edges {
            if d[e.u] != i32::MAX && d[e.u] + e.w < d[e.v] { d[e.v] = d[e.u] +
e.w; }
        }
    }
    // Checar ciclos negativos...
    Some(d)
}

fn kmp_matcher(t: &str, p: &str) {

    let mut q = 0;

    }
}

fn b_tree_search(node: &BNode, k: i32) -> Option<(&BNode, usize)> {
    let mut i = 0;
    while i < node.n && k > node.keys[i] { i += 1; }
    if i < node.n && k == node.keys[i] { return Some((node, i)); }
    if node.leaf { None } else { b_tree_search(&node.children[i], k) }
}
