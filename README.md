# SysP

SysP é uma linguagem de sistemas compilada.

Características:

- compilador em C++23
- arquitetura baseada em grafo
- módulos organizados em categorias
- geração de executável nativo

Estrutura do compilador:

frontend → lexer / parser / AST  
middleend → IR / CFG / Metatron / optimizer  
backend → x86 / arm
